# 数据生成器

用于生成相关数据库数据

包括：

- 生成一份随机的用户数据
  - 个人信息
    - 用户名：字符串
    - 邮箱：字符串
    - 电话：字符串
    - 自我介绍：字符串
    - 密码统一为 123456 的 hash
  - 购物车：若干个商品 ID
  - 订单列表：若干个订单 ID
    - 订单 ID
    - 商品 ID
    - 数量
    - 价格
    - 状态 ['待付款', '待发货', '待收货', '已完成', '已取消']
- 生成一份随机的商品数据
  - 商品信息
    - 商品 ID (UUID)
    - 商品名
    - 特点
    - 价格
    - 介绍
    - 图片
      - 一张缩略图
      - 两个详情图
